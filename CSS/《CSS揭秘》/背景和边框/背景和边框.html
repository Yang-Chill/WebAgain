伪随机背景

通过重复平铺，把不同尺寸、间距和颜色的贴片叠加起来，重复若干次后会再次统一对齐

平铺间距越大的贴片，放在越高层

对齐时所平铺的总长=每种贴片的长度的最小公倍数

为使随机性更真实，应让最小公倍数最大化，长度尽可能选择质数
